# Jiger

Editor

#### Prerequisite
- C++
- SFML 2.5 (Graphics library)

#### Aim
- Should be fast
- Good UI
- Customizable

#### Build and Run instruction
##### Linux (Ubuntu)
To build
```
make compile2.5
```

To run
```
make run2.5
```

Refer Makefile. Commands with `compile` and `run` won't work without append `2.5` with them.
